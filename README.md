# Movie Review Classification with Naive Bayes

A machine learning project that builds a Naive Bayes classifier to predict movie review sentiment using natural language processing techniques.

## Overview

This project preprocesses movie review text, extracts useful features, and applies the Naive Bayes algorithm to classify reviews as positive or negative.

## Technologies Used

- Python
- NumPy
- pandas
- matplotlib
- WordCloud
- Jupyter Notebook

## Features

- Text preprocessing
- Stopword removal
- Feature extraction
- Naive Bayes classification
- Sentiment prediction
- Data visualization
